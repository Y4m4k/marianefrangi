# Mariane Frangi Author Website

Astro + Cloudflare Pages author website.

## Where to edit

Most content is in:

- `src/components/Hero.astro`
- `src/components/Books.astro`
- `src/components/About.astro`
- `src/components/Visits.astro`
- `src/components/Activities.astro`
- `src/components/Events.astro`
- `src/components/Contact.astro`

The homepage structure is in:

- `src/pages/index.astro`

The design/colors are in:

- `src/styles/global.css`

Images go in:

- `public/images/`

Downloads go in:

- `public/downloads/`

## Replace these placeholders

Search the project for:

- `YOUR-EMAIL-HERE`
- `PASTE-AMAZON-LINK-HERE`
- `PASTE-YOUR-MAIN-AMAZON-AUTHOR-LINK-HERE`
- `PASTE-AMAZON-AUTHOR-LINK-HERE`

Replace them with real links.

## Cloudflare build settings

Framework preset: Astro  
Build command: `npm run build`  
Build output directory: `dist`

## Local development

```bash
npm install
npm run dev
```

Then open the local URL shown in your terminal.

## Deploy

Commit changes to GitHub. Cloudflare Pages will deploy automatically.
